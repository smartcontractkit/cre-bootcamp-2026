## Prerequisitos Importantes

Para aprovechar al máximo este bootcamp, te recomendamos tener lo siguiente preparado **antes del Día 1**. Parte de esto se cubrirá brevemente para poder dedicar más tiempo al desarrollo del proyecto.

### Configuración Requerida

- **Node.js v20 o superior** - [Descargar aquí](https://nodejs.org/)
- **Bun v1.3 o superior** - [Descargar aquí](https://bun.sh/docs/installation)
- **CRE CLI** - [Instrucciones de instalación](https://docs.chain.link/cre/getting-started/cli-installation)
- **Foundry** - [Instrucciones de instalación](https://book.getfoundry.sh/getting-started/installation)
- **Agregar la red Ethereum Sepolia a tu wallet** - [Agregar red aquí](https://chainlist.org/chain/11155111)
- **Obtener ETH de Ethereum Sepolia del faucet** - [Chainlink Faucet](https://faucets.chain.link/sepolia)
- **Clave API de Gemini LLM** - [Obtener desde Google AI Studio](https://aistudio.google.com/apikey)

### Opcional
